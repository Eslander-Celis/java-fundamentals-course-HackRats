# Fundamentos de Java para Principiantes

## Resumen del curso

Este curso introduce los fundamentos de la programacion con Java para estudiantes de secundaria de 12 a 17 anos sin experiencia previa. La propuesta esta organizada en seis lecciones en video, con una duracion total aproximada de 60 minutos, y desarrolla los conceptos de forma progresiva hasta llegar a un proyecto final basado en programacion orientada a objetos.

El curso no requiere instalacion de herramientas. Las practicas se realizan en editores gratuitos en linea, accesibles desde un navegador web como Chrome, Firefox, Safari o Edge.

**Duracion total:** 60 minutos aproximadamente  
**Numero de lecciones:** 6 videos  
**Publico objetivo:** estudiantes de secundaria sin experiencia previa en programacion  
**Prerrequisitos:** ninguno  
**Herramientas:** JDoodle, OnlineGDB, Replit y GitHub  
**Repositorio de codigo fuente:** [java-fundamentals-course-HackRats](https://github.com/Eslander-Celis/java-fundamentals-course-HackRats)

## Objetivos de aprendizaje

Al finalizar el curso, los estudiantes seran capaces de:

- Comprender que es la programacion y como se usa Java para crear programas.
- Ejecutar programas Java desde un editor en linea.
- Utilizar variables, tipos de datos, operadores y expresiones.
- Aplicar estructuras condicionales y bucles.
- Crear metodos simples con parametros.
- Leer informacion desde consola usando `Scanner`.
- Comprender clases, objetos, atributos, constructores y metodos.
- Aplicar encapsulacion mediante atributos privados, getters y setters.
- Manipular arreglos simples en un programa Java.
- Desarrollar una aplicacion basica orientada a objetos.

## Leccion 1: Introduccion a la programacion y Java

La primera leccion presenta la programacion como una forma de escribir instrucciones ordenadas para que una computadora realice una tarea. Se explica el concepto de algoritmo mediante ejemplos cotidianos y se introduce Java como lenguaje de programacion utilizado en aplicaciones empresariales, servicios web, sistemas bancarios, aplicaciones moviles y otros productos de software.

El estudiante ejecuta su primer programa en Java con `System.out.println()` y modifica el mensaje mostrado en consola.

**Conceptos principales:**

- Programacion.
- Algoritmos.
- Java.
- Clase principal.
- Metodo `main`.
- Salida por consola.

**Actividad:** modificar el mensaje del programa "Hola Mundo" para mostrar un saludo personalizado.

## Leccion 2: Variables, tipos de datos y operadores

La segunda leccion explica como almacenar informacion en un programa mediante variables. Se presentan tipos de datos basicos de Java y se muestra como combinarlos con operadores aritmeticos para resolver ejercicios simples.

**Conceptos principales:**

- Variables.
- Tipo `int`.
- Tipo `double`.
- Tipo `String`.
- Tipo `boolean`.
- Operadores aritmeticos.
- Expresiones.

**Actividad:** crear una ficha personal que muestre nombre, edad, curso favorito y promedio de notas.

## Leccion 3: Condicionales y bucles

La tercera leccion desarrolla estructuras de control que permiten tomar decisiones y repetir instrucciones. Se trabaja con condicionales `if` y `else`, operadores relacionales y bucles `for` y `while`.

**Conceptos principales:**

- Condicional `if`.
- Alternativa `else`.
- Operadores relacionales.
- Bucle `for`.
- Bucle `while`.

**Actividad:** crear un programa que muestre la tabla de multiplicar del numero 5.

## Leccion 4: Metodos, parametros y entrada por consola

La cuarta leccion presenta los metodos como bloques reutilizables de codigo. Se explica como declarar metodos, enviar parametros, retornar valores simples y leer informacion ingresada por el usuario mediante `Scanner`.

**Conceptos principales:**

- Metodos.
- Parametros.
- Valores de retorno.
- Entrada por consola.
- Clase `Scanner`.

**Actividad:** crear un metodo llamado `presentarse` que reciba el nombre y la edad del estudiante, y muestre un mensaje completo en consola.

## Leccion 5: Introduccion a la programacion orientada a objetos

La quinta leccion introduce los fundamentos de la programacion orientada a objetos. Se explica la diferencia entre clase y objeto, el uso de atributos y metodos, la creacion de constructores y la aplicacion basica de encapsulacion con getters y setters.

**Conceptos principales:**

- Clases.
- Objetos.
- Atributos.
- Metodos.
- Constructores.
- Encapsulacion.
- Getters y setters.

**Actividad:** crear una clase `Mascota` con nombre, edad, constructor, metodos de acceso y un metodo para presentarse.

## Leccion 6: Proyecto final integrador

La sexta leccion integra los contenidos del curso mediante un programa basico de registro de estudiantes. El proyecto permite ingresar datos, crear objetos, almacenarlos en un arreglo y mostrar informacion en consola.

**Conceptos aplicados:**

- Variables y tipos de datos.
- Condicionales.
- Bucles.
- Metodos.
- Entrada por consola.
- Clases y objetos.
- Constructores.
- Encapsulacion.
- Arreglos.

**Proyecto:** registro basico de estudiantes con nombre y edad. El programa muestra los datos registrados e identifica si cada estudiante se encuentra dentro del rango principal de edad escolar secundaria.

El equipo HackRats desarrollo una ruta de aprendizaje introductoria para estudiantes de secundaria, organizada en seis lecciones progresivas. La propuesta inicia con conceptos basicos de programacion y Java, continua con estructuras fundamentales del lenguaje y finaliza con un proyecto integrador basado en programacion orientada a objetos.
