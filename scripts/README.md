# Guiones de las lecciones

Esta carpeta contiene los guiones finales utilizados en los videos del curso **Fundamentos de Java para Principiantes**.

Cada guion desarrolla la narracion de una leccion y mantiene una secuencia progresiva para estudiantes de secundaria sin experiencia previa en programacion. Las practicas se realizan en editores Java gratuitos y en linea, por lo que los estudiantes solo necesitan un navegador web actualizado.

| 2 | [lesson-02-variables.md](lesson-02-variables.md) | Variables, tipos de datos, operadores y expresiones |
| 3 | [lesson-03-condicionales-bucles.md](lesson-03-condicionales-bucles.md) | Condicionales y bucles |
| 4 | [lesson-04-metodos.md](lesson-04-metodos.md) | Metodos, parametros y entrada por consola |

| 6 | [lesson-06-proyecto-final.md](lesson-06-proyecto-final.md) | Proyecto final integrador |
