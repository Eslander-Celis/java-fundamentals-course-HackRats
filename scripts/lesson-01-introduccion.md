# Leccion 1: Introduccion a la programacion y Java

Hola a todos. Bienvenidos al curso Fundamentos de Java para Principiantes.

En este curso vamos a aprender a programar desde cero usando Java. No necesitas tener experiencia previa y tampoco necesitas instalar programas en tu computadora. Todas las practicas se realizaran desde el navegador, usando herramientas gratuitas en linea.

Para empezar, pensemos en una situacion cotidiana. Si queremos preparar un sandwich, seguimos una serie de pasos: tomar el pan, agregar los ingredientes, cerrar el sandwich y servirlo. Si los pasos estan desordenados o falta uno, el resultado puede cambiar.

En programacion ocurre algo parecido. Programar significa escribir instrucciones claras y ordenadas para que una computadora realice una tarea. A esa secuencia de pasos se le llama algoritmo.

Durante este curso usaremos Java. Java es un lenguaje de programacion muy utilizado en el desarrollo de aplicaciones empresariales, sistemas bancarios, servicios web, aplicaciones moviles y otros productos de software.

En esta primera leccion ejecutaremos nuestro primer programa en Java. El objetivo es mostrar un mensaje en pantalla usando la consola.

El programa se llama "Hola Mundo" y es uno de los primeros ejercicios que suelen realizar las personas que empiezan a programar.

El codigo tiene una clase principal llamada `Main`. Dentro de esa clase se encuentra el metodo `main`, que es el punto desde donde inicia la ejecucion del programa. Luego usamos la instruccion `System.out.println()` para mostrar un mensaje en pantalla.

Cuando ejecutamos el programa, la consola muestra el texto "Hola Mundo". Esto confirma que el codigo se escribio correctamente y que Java pudo ejecutarlo.

Ahora realizamos una pequena modificacion. En lugar de mostrar "Hola Mundo", cambiamos el mensaje por un saludo personalizado, por ejemplo: "Hola, soy Ana".

Con este cambio, el estudiante comprueba que puede modificar un programa y observar un resultado diferente. Aunque es un ejercicio pequeno, representa el primer paso para crear programas mas completos.

En esta leccion aprendimos que programar es dar instrucciones, que un algoritmo es una secuencia de pasos y que Java permite escribir programas que la computadora puede ejecutar.

En la siguiente leccion aprenderemos a usar variables, que sirven para guardar informacion dentro de un programa.
