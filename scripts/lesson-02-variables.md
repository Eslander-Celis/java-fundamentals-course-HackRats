# Leccion 2: Variables, tipos de datos y operadores

Hola a todos. En esta segunda leccion aprenderemos a usar variables en Java.

Una variable es un espacio donde un programa puede guardar informacion. Podemos imaginarla como una caja con una etiqueta. Por ejemplo, podemos tener una caja llamada `nombre` para guardar un texto, una caja llamada `edad` para guardar un numero entero o una caja llamada `promedio` para guardar un numero decimal.

Java necesita saber que tipo de informacion guardara cada variable. Por eso usamos tipos de datos.

El tipo `int` se utiliza para numeros enteros, como una edad o una cantidad. El tipo `double` se utiliza para numeros decimales, como una altura o un promedio. El tipo `String` se utiliza para texto, como un nombre o una frase. El tipo `boolean` se utiliza para valores de verdadero o falso.

Por ejemplo, si queremos guardar los datos de un estudiante, podemos crear una variable para su nombre, otra para su edad y otra para su curso favorito.

Luego podemos mostrar esos datos en consola usando `System.out.println()`.

Ademas de guardar informacion, Java permite realizar operaciones. Podemos sumar, restar, multiplicar y dividir valores numericos. Esto es util para resolver ejercicios simples, como calcular el promedio de dos notas.

Por ejemplo, si tenemos una primera nota de 18 y una segunda nota de 17, podemos sumarlas y dividir el resultado entre 2. El programa guardara el promedio en una variable y luego lo mostrara en pantalla.

La actividad de esta leccion consiste en crear una ficha personal. El programa debe mostrar el nombre del estudiante, su edad, su curso favorito y el promedio de dos notas.

Con esta practica, el estudiante aprende que un programa puede guardar datos, procesarlos y mostrar resultados.

En resumen, en esta leccion vimos variables, tipos de datos y operadores. Estos conceptos son fundamentales porque casi todos los programas necesitan almacenar y trabajar con informacion.
