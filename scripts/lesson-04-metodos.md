# Leccion 4: Metodos, parametros y entrada por consola

Hola a todos. En esta leccion aprenderemos a usar metodos en Java.

Un metodo es un bloque de codigo que realiza una tarea especifica. Los metodos ayudan a organizar mejor un programa, evitan repetir codigo y hacen que las instrucciones sean mas faciles de entender.

Por ejemplo, si queremos saludar a varias personas, no necesitamos escribir todo el mensaje una y otra vez. Podemos crear un metodo llamado `saludar` y usarlo cada vez que lo necesitemos.

Los metodos tambien pueden recibir informacion. A esa informacion se le llama parametro. Si un metodo recibe un nombre como parametro, puede usar ese valor para mostrar un saludo personalizado.

Ademas, algunos metodos pueden devolver un resultado. Por ejemplo, un metodo llamado `sumar` puede recibir dos numeros, sumarlos y devolver el resultado.

En esta leccion tambien veremos como leer datos escritos por el usuario. Para eso usamos la clase `Scanner`.

`Scanner` permite que el programa reciba informacion desde la consola. Por ejemplo, podemos pedir que el estudiante escriba su nombre y su edad. Luego el programa puede usar esos datos para mostrar un mensaje.

La actividad de esta leccion consiste en crear un metodo llamado `presentarse`. Este metodo recibira dos datos: el nombre y la edad del estudiante. Luego mostrara un mensaje como: "Hola, soy Daniela y tengo 15 anos".

Con esta practica, el estudiante combina metodos, parametros y entrada por consola.

Este tema es importante porque los metodos permiten dividir un problema grande en partes mas pequenas. Asi, el codigo queda mas ordenado y es mas facil de modificar.

En resumen, en esta leccion aprendimos que los metodos sirven para agrupar instrucciones, que los parametros permiten enviar informacion a un metodo y que `Scanner` permite leer datos ingresados por el usuario.
