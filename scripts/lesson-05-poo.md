# Leccion 5: Programacion orientada a objetos

Hola a todos. En esta leccion aprenderemos los conceptos iniciales de la programacion orientada a objetos.

La programacion orientada a objetos permite representar elementos del mundo real dentro de un programa. Por ejemplo, podemos representar un estudiante, una mascota, un auto o un videojuego.

Para hacer esto usamos clases y objetos.

Una clase es como un molde. Define las caracteristicas y comportamientos que tendran los objetos creados a partir de ella. Por ejemplo, una clase `Estudiante` puede tener atributos como nombre y edad, y metodos como saludar o mostrar sus datos.

Un objeto es una instancia de una clase. Si la clase es el molde, el objeto es un elemento creado a partir de ese molde. Por ejemplo, podemos crear un estudiante llamado Carlos con 15 anos.

Los atributos representan las caracteristicas del objeto. Los metodos representan las acciones que el objeto puede realizar.

Tambien veremos constructores. Un constructor se ejecuta cuando se crea un objeto y sirve para asignar valores iniciales. Por ejemplo, al crear una mascota podemos indicar su nombre y edad desde el inicio.

Otro concepto importante es la encapsulacion. Encapsular significa proteger los datos internos de una clase. Para lograrlo, declaramos los atributos como privados usando `private` y accedemos a ellos mediante metodos conocidos como getters y setters.

Los getters permiten obtener el valor de un atributo. Los setters permiten modificarlo de forma controlada.

La actividad de esta leccion consiste en crear una clase llamada `Mascota`. Esta clase tendra nombre, edad, un constructor, metodos getter y setter, y un metodo para que la mascota se presente.

Con esta practica, el estudiante comprende como una clase puede representar un elemento real con datos y comportamientos.

En resumen, en esta leccion aprendimos que las clases funcionan como moldes, los objetos son elementos creados a partir de esos moldes, los atributos guardan informacion, los metodos realizan acciones y la encapsulacion ayuda a proteger los datos.
